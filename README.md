# GradeOps 🚀

> An AI-powered, full-stack grading and automated feedback SaaS application.

GradeOps streamlines the evaluation process by leveraging generative AI to assess student submissions, provide constructive feedback, and manage academic data efficiently. Built with modern web technologies, it bridges the gap between scalable full-stack architecture and practical AI integration.

## ✨ Features

*   **AI-Powered Grading:** Integrates the Gemini API to analyze and evaluate student submissions with high accuracy.
*   **Automated Feedback Dispatch:** Utilizes Nodemailer to instantly send personalized evaluation reports to students.
*   **Student Query Interface:** A dedicated dashboard for students to view grades and interact with their feedback.
*   **Robust Data Management:** Built on a relational SQL database (MySQL) managed via Prisma for seamless data handling and integrity.
*   **Responsive UI:** A clean, modern interface designed with Next.js and Tailwind CSS.

## 🛠️ Tech Stack

**Frontend:**
*   Next.js
*   Tailwind CSS

**Backend & Database:**
*   Next.js API Routes
*   MySQL
*   Prisma ORM
*   Nodemailer

**AI & Machine Learning:**
*   Google Gemini API
